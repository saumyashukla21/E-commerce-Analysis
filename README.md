# E-commerce-Analysis
# VIVO CALIF Sales Dashboard

## Overview
This project is an interactive **Sales Dashboard** created in **Microsoft Excel** to analyze sales performance, customer behavior, product trends, and delivery metrics. The dashboard provides a comprehensive view of key business insights through charts, maps, and KPIs.

## Dashboard Preview

![Dashboard Screenshot](E-Commerce%20Dashboard.png)

## Built With
- **Microsoft Excel**
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Conditional Formatting
- Maps
- Formulas & Data Analysis

## Key Features

### 📈 Sales Performance
- Total Orders
- Total Quantity Sold
- Total Revenue
- Weekly Sales Trend (Last 13 Weeks)

### 👥 Customer Insights
- Gender-wise purchase distribution
- Order value per customer
- Customer ratings analysis

### 🛍️ Product Analysis
- Popular products ranking
- Gender-wise product purchases

### 🚚 Delivery Analytics
- Average delivery time
- Delivery time distribution

### 🌎 Geographic Analysis
- State-wise quantity sold across California

### 🎛️ Interactive Filters
Users can filter the dashboard by:
- Order Mode
  - App
  - Instagram
  - Partner App
  - Target.com
  - Website
- Gender
  - Female
  - Male
  - Other
  - Unknown
